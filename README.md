 ## 📁 프로젝트 구조

 ### Frontend 구조
 
 ```bash
frontend
├── public                # index.html / favicon
└── src
    ├── api               # api 관련 모듈들
    ├── assets            # 필요한 리소스들
    ├── components        # 컴포넌트들
    ├── router            # 라우터들
    ├── store             # 상태관리
    ├── styles            # 공통 스타일들(base.scss)
    ├── utils             # 재사용 가능한 모듈들
    ├── views             # 페이지들
    ├── App.vue(FILE)     # App 컴포넌트
    └── main.js(FILE)     # entry point
```


 ## ▶️ 설치 및 실행
 
 ### 프로젝트 설치 및 실행 - Development mode
 #### client
.env 파일
```bash
VUE_APP_API_URLT=YOUR SERVER API
 ```
 실행
 ```bash
$ git clone https://github.com/seggle/FE.git
$ cd FE
$ npm install
$ npm run serve
 ```
